## 📖 Overview
This repository showcases hands-on implementation of **R programming** for solving data science problems, building machine learning models, and creating insightful visualizations.  

It represents a **learning-driven exploration**, covering a wide range of concepts rather than focusing on a single project.

## 📁 Notebooks

- 📘 **R.ipynb**  
  Comprehensive workflows including data manipulation, modeling, and visualization  

- 📗 **R_Notebook.ipynb**

  KNN and clustering methods with evaluation techniques
